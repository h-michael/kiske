kiske
========


`kiske` is a zsh theme.

## Features

- git prompt
- ruby, node, python, go version at current directory

## Installation

By using [zplug](https://github.com/zplug/zplug) you can easily install as a theme:

```zsh
zplug "h-michael/kiske", as:theme
```
